# Movie Catalog API

Esta é uma API REST para listagem de filmes desenvolvida em Java com o framework Spring Boot.
Este projeto foi construído por mim do absoluto zero para aprender os conceitos fundamentais do desenvolvimento de servidores (Backend).

# O que este projeto faz?
* Define um modelo de dados estruturado para um Filme (Título, Gênero e Avaliação).
* Cria uma rota de internet (Endpoint) para listar os filmes salvos na memória.
* Devolve os dados organizados no formato universal **JSON**.

# Tecnologias Utilizadas
* Java 21
* Spring Boot 3 (Framework para criação rápida de APIs)
* Maven/Gradle (Gerenciador de dependências do projeto)

# Como acessar a API?
Com o projeto rodando no seu computador, você pode acessar a lista de filmes abrindo o navegador no endereço:
`http://localhost:8080/filmes`

# Aprendizados adquiridos neste projeto
* Configuração do motor do Java (SDK) em uma IDE profissional.
* Uso de boas práticas de segurança com variáveis privadas e métodos Getters e Setters.
* Criação de controladores web utilizando anotações como `@RestController` e `@GetMapping`.
* Compreensão e leitura do formato de dados JSON.
